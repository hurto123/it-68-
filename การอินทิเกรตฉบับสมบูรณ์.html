ภาพรวมการอินทิเกรตในแคลคูลัส (ฉบับสมบูรณ์)

คู่มือเรียนรู้และทบทวนแนวคิดการอินทิเกรต ตั้งแต่พื้นฐานจนถึงการประยุกต์ใช้

การอินทิเกรตคือหนึ่งในสองแนวคิดหลักของแคลคูลัส โดยแก่นแท้แล้ว การอินทิเกรตมีสองความหมายหลักที่เชื่อมโยงกันอย่างน่าทึ่งผ่าน ทฤษฎีบทมูลฐานของแคลคูลัส

  1. การดำเนินการผกผันของการหาอนุพันธ์ (Antiderivative): เป็นกระบวนการ "ย้อนกลับ" ของการหาอนุพันธ์ ถ้าเรารู้อัตราการเปลี่ยนแปลง (เช่น ความเร็ว) เราสามารถหาปริมาณตั้งต้นได้ (เช่น ตำแหน่ง)
  2. การหาปริมาณสะสม (Accumulation): เป็นกระบวนการ "บวกรวม" ค่าเล็กๆ ที่มีจำนวนเป็นอนันต์เข้าด้วยกัน ซึ่งมักจะถูกอธิบายในรูปของ การหาพื้นที่ใต้กราฟ

ส่วนที่ 1: ปริพันธ์ไม่จำกัดเขต (Indefinite Integral)

นี่คือมุมมองของการอินทิเกรตในฐานะ "ปฏิยานุพันธ์" (Antiderivative)

$$ \int f(x) \,dx = F(x) + C $$
  • \(F(x)\) คือฟังก์ชันที่เมื่อหาอนุพันธ์แล้วจะได้ \(f(x)\) (นั่นคือ \(F'(x) = f(x)\))
  • \(f(x)\) คือ ตัวถูกอินทิเกรต (Integrand)
  • \(C\) คือ ค่าคงที่ของการอินทิเกรต (Constant of Integration)

คำอธิบายเชิงลึก: ทำไมต้องมี +C?

ค่า \(+C\) ไม่ได้เป็นเพียงตัวอักษรที่ต้องเติมท้ายคำตอบ แต่มีความหมายในเชิงกราฟที่สำคัญอย่างยิ่ง:

  • ครอบครัวของฟังก์ชัน: \(F(x) + C\) ไม่ได้หมายถึงฟังก์ชันเดียว แต่หมายถึง "ครอบครัวของฟังก์ชัน" ที่มีค่า C แตกต่างกันไป เช่น \(x^2\), \(x^2 + 5\), \(x^2 - 100\) ล้วนเป็นสมาชิกของครอบครัว \(x^2 + C\)
  • ความชันเท่ากัน: กราฟของทุกฟังก์ชันในครอบครัวนี้จะมีรูปร่างเหมือนกันทุกประการ เพียงแต่ถูกเลื่อนขึ้นหรือลงในแนวดิ่งเท่านั้น ซึ่งหมายความว่า ณ จุด \(x\) ใดๆ ก็ตาม ความชันของเส้นสัมผัสกราฟ (ซึ่งก็คือค่าของอนุพันธ์) ของทุกๆ ฟังก์ชันในครอบครัวนี้จะเท่ากันเสมอ
  • การสูญเสียข้อมูล: เมื่อเราหาอนุพันธ์ เช่น \(\frac{d}{dx}(x^2 + 5) = 2x\) ข้อมูลเกี่ยวกับค่าคงที่ \(+5\) จะหายไป (เพราะอนุพันธ์ของค่าคงที่คือ 0) ดังนั้น เมื่อเราอินทิเกรต \(2x\) กลับไป เราจึงต้องใส่ \(+C\) เพื่อแสดงถึงข้อมูลค่าคงที่ที่อาจจะเคยมีอยู่และได้สูญหายไป

ที่มาของสูตรอินทิเกรต

1. สูตรกำลัง (Power Rule)

$$ \int x^n \,dx = \frac{x^{n+1}}{n+1} + C, \quad (n \neq -1) $$

คำอธิบาย: เราต้องการฟังก์ชันที่เมื่อ "ดิฟ" แล้วเลขชี้กำลังลดลง 1 ดังนั้นฟังก์ชันตั้งต้นต้องมีเลขชี้กำลังมากกว่าอยู่ 1 คือ \(x^{n+1}\) แต่เมื่อเราลองดิฟ \(\frac{d}{dx}(x^{n+1}) = (n+1)x^n\) จะเห็นว่ามี \((n+1)\) เกินมา เราจึงต้องหารด้วย \((n+1)\) เพื่อหักล้างกัน กลายเป็น \(\frac{d}{dx}\left[ \frac{x^{n+1}}{n+1} \right] = x^n\)

2. กรณี n = -1

$$ \int \frac{1}{x} \,dx = \ln|x| + C $$

คำอธิบาย: ทำไมต้องเป็น \(\ln|x|\)?

  • โดเมน: ฟังก์ชัน \(f(x) = 1/x\) สามารถหาค่าได้ทั้ง \(x\) ที่เป็นบวกและลบ (ยกเว้น x=0) แต่ฟังก์ชัน \(\ln(x)\) โดยปกติจะนิยามสำหรับ \(x > 0\) เท่านั้น
  • การใช้ค่าสัมบูรณ์: การใช้ \(\ln|x|\) ทำให้โดเมนของคำตอบสอดคล้องกับโดเมนของคำถาม
    • เมื่อ \(x > 0\), \(\frac{d}{dx}(\ln x) = \frac{1}{x}\)
    • เมื่อ \(x < 0\), \(\frac{d}{dx}(\ln(-x)) = \frac{1}{-x} \cdot (-1) = \frac{1}{x}\) (ตามกฎลูกโซ่)
  • ดังนั้น \(\ln|x|\) คือปฏิยานุพันธ์ของ \(1/x\) ที่สมบูรณ์ที่สุด

3. ฟังก์ชันตรีโกณมิติเพิ่มเติม

$$ \int \tan x \,dx = \ln|\sec x| + C $$

ที่มา: แปลง \(\tan x = \frac{\sin x}{\cos x}\) แล้วใช้วิธีแทนค่าโดยให้ \(u = \cos x\) ดังนั้น \(du = -\sin x \,dx\) จะได้ \(\int \frac{-1}{u} \,du = -\ln|u| + C = -\ln|\cos x| + C = \ln|\sec x| + C\)

$$ \int \sec x \,dx = \ln|\sec x + \tan x| + C $$

ที่มา: ใช้เทคนิคคูณด้วย \(\frac{\sec x + \tan x}{\sec x + \tan x}\) แล้วให้ \(u = \sec x + \tan x\) จะพบว่า \(du = (\sec x \tan x + \sec^2 x) \,dx\) ซึ่งคือตัวเศษพอดี อินทิกรัลจึงกลายเป็น \(\int \frac{1}{u} \,du = \ln|u| + C\)

4. สูตรที่ให้ผลลัพธ์เป็นฟังก์ชันตรีโกณมิติผกผัน

$$ \int \frac{1}{\sqrt{a^2 - x^2}} \,dx = \arcsin\left(\frac{x}{a}\right) + C $$

ที่มา (Trigonometric Substitution):

  1. รูปแบบ \(a^2 - x^2\) สอดคล้องกับเอกลักษณ์ \(1 - \sin^2\theta = \cos^2\theta\)
  2. ให้ \(x = a \sin \theta\), ดังนั้น \(dx = a \cos \theta \,d\theta\)
  3. ตัวส่วนจะกลายเป็น \(\sqrt{a^2 - a^2\sin^2\theta} = \sqrt{a^2\cos^2\theta} = a \cos \theta\)
  4. อินทิกรัลจะกลายเป็น \(\int \frac{1}{a \cos \theta} \cdot (a \cos \theta \,d\theta) = \int 1 \,d\theta = \theta + C\)
  5. จาก \(x = a \sin \theta\) จะได้ \(\theta = \arcsin\left(\frac{x}{a}\right)\) แทนค่ากลับได้คำตอบ
$$ \int \frac{1}{a^2 + x^2} \,dx = \frac{1}{a}\arctan\left(\frac{x}{a}\right) + C $$

ที่มา (Trigonometric Substitution):

  1. รูปแบบ \(a^2 + x^2\) สอดคล้องกับเอกลักษณ์ \(1 + \tan^2\theta = \sec^2\theta\)
  2. ให้ \(x = a \tan \theta\), ดังนั้น \(dx = a \sec^2\theta \,d\theta\)
  3. อินทิกรัลจะกลายเป็น \(\int \frac{1}{a^2(1+\tan^2\theta)} \cdot (a \sec^2\theta \,d\theta) = \int \frac{1}{a^2\sec^2\theta} \cdot (a \sec^2\theta \,d\theta) = \int \frac{1}{a} \,d\theta = \frac{1}{a}\theta + C\)
  4. จาก \(x = a \tan \theta\) จะได้ \(\theta = \arctan\left(\frac{x}{a}\right)\) แทนค่ากลับได้คำตอบ

ส่วนที่ 2: ปริพันธ์จำกัดเขต (Definite Integral)

นี่คือมุมมองของการอินทิเกรตในฐานะ "การรวมกันของค่าอนันต์" ซึ่งแสดงภาพเป็นการหาพื้นที่ใต้กราฟ

ที่มาของแนวคิด: ผลบวกรีมันน์ (Riemann Sum)

แนวคิดเริ่มต้นคือการ "ประมาณค่า" พื้นที่ใต้เส้นโค้งด้วย สี่เหลี่ยมผืนผ้า หลายๆ แท่ง

  1. แบ่ง: แบ่งช่วง \([a, b]\) บนแกน x ออกเป็น \(n\) ช่วงย่อยๆ ที่มีความกว้าง \(\Delta x = \frac{b-a}{n}\)
  2. สร้างสี่เหลี่ยม: ในแต่ละช่วงย่อย สร้างสี่เหลี่ยมผืนผ้าที่สูงเท่ากับค่าของฟังก์ชัน ณ จุดใดจุดหนึ่ง
  3. รวม: คำนวณพื้นที่ของสี่เหลี่ยมแต่ละแท่ง (\(f(x_i^*) \times \Delta x\)) แล้วนำมารวมกันทั้งหมด: \(\sum f(x_i^*) \Delta x\)
  4. ใช้ลิมิต: การประมาณค่าจะแม่นยำขึ้นเมื่อเราใช้จำนวนสี่เหลี่ยมมากขึ้น (ทำให้ \(\Delta x\) แคบลง) ถ้าเราใช้จำนวนสี่เหลี่ยมเป็นอนันต์ (\(n \to \infty\)) ผลรวมนี้จะลู่เข้าสู่ค่าที่แท้จริงของพื้นที่ใต้กราฟ
นิยาม: $$ \int_{a}^{b} f(x) \,dx = \lim_{n \to \infty} \sum_{i=1}^{n} f(x_i^*) \Delta x $$

ทฤษฎีบทมูลฐานของแคลคูลัส (FTC)

ทฤษฎีบทนี้คือสะพานเชื่อมที่ยิ่งใหญ่ที่สุดในแคลคูลัส มันเชื่อมโยงแนวคิด "ปฏิยานุพันธ์" (พีชคณิต) กับ "พื้นที่ใต้กราฟ" (เรขาคณิต) เข้าด้วยกัน

  • FTC ส่วนที่ 1: อธิบายว่าการอินทิเกรตและการหาอนุพันธ์เป็นกระบวนการผกผันต่อกัน ถ้าเราสร้าง "ฟังก์ชันสะสมพื้นที่" \(g(x) = \int_{a}^{x} f(t) \,dt\) แล้วอัตราการเปลี่ยนแปลงของพื้นที่นี้ ณ จุด x จะเท่ากับความสูงของกราฟ ณ จุดนั้นพอดี: \(g'(x) = f(x)\)
  • FTC ส่วนที่ 2: ให้วิธีคำนวณปริพันธ์จำกัดเขตที่ง่ายและทรงพลัง โดยไม่ต้องคำนวณลิมิตของผลบวกรีมันน์ที่ยุ่งยาก
$$ \int_{a}^{b} f(x) \,dx = F(b) - F(a) $$

ส่วนที่ 3: เทคนิคการอินทิเกรต

สำหรับฟังก์ชันที่ซับซ้อน เราไม่สามารถใช้สูตรพื้นฐานได้โดยตรง จึงต้องมีเทคนิคต่างๆ เข้ามาช่วย

1. การอินทิเกรตโดยการแทนค่า (Integration by Substitution)

แนวคิด: เป็นการย้อนกลับของ "กฎลูกโซ่" หัวใจสำคัญคือการมองหา "ฟังก์ชันตัวใน" (\(u\)) และ "อนุพันธ์ของมัน" (\(du\)) ที่ปรากฏเป็นผลคูณอยู่ในอินทิกรัล

สำหรับปริพันธ์จำกัดเขต: เมื่อใช้เทคนิคนี้กับ \(\int_{a}^{b} f(g(x))g'(x) \,dx\) มี 2 วิธีให้เลือก:

  1. เปลี่ยนขอบเขต: เมื่อเปลี่ยนเป็นตัวแปร \(u\) ให้เปลี่ยนขอบเขตจาก \(x=a, x=b\) เป็น \(u=g(a), u=g(b)\) แล้วคำนวณในเทอมของ \(u\) ให้เสร็จสิ้นโดยไม่ต้องแทนค่ากลับ
  2. ไม่เปลี่ยนขอบเขต: คำนวณปริพันธ์ไม่จำกัดเขตให้เสร็จสิ้นในเทอมของ \(u\) ก่อน แล้วแทนค่ากลับเป็น \(x\) จากนั้นจึงใช้ขอบเขตเดิมคือ \(a\) และ \(b\)

2. การอินทิเกรตโดยการแยกส่วน (Integration by Parts)

$$ \int u \,dv = uv - \int v \,du $$

แนวคิด: เป็นการย้อนกลับของ "กฎผลคูณ" ใช้เมื่อตัวถูกอินทิเกรตอยู่ในรูปผลคูณของสองฟังก์ชันที่ไม่เกี่ยวข้องกัน

หลักการเลือก u: ใช้หลัก LIATE เพื่อช่วยตัดสินใจว่าฟังก์ชันไหนควรเป็น \(u\)

  • L - Logarithmic (เช่น \(\ln x\))
  • I - Inverse Trigonometric (เช่น \(\arcsin x\))
  • A - Algebraic (เช่น \(x^2, 3x\))
  • T - Trigonometric (เช่น \(\sin x, \cos x\))
  • E - Exponential (เช่น \(e^x, 2^x\))

กรณีวนลูป (Cyclic): สำหรับโจทย์เช่น \(I = \int e^x\cos(x) \,dx\) เมื่อทำ By Parts สองครั้ง อินทิกรัลจะวนกลับมาเป็นรูปแบบเดิม เราสามารถย้ายข้างสมการเพื่อแก้หาค่า I ได้

3. การอินทิเกรตโดยการแยกเศษส่วนย่อย (Integration by Partial Fractions)

แนวคิด: ใช้สำหรับอินทิเกรตฟังก์ชันตรรกยะ (\(P(x)/Q(x)\)) โดยการแยกฟังก์ชันเศษส่วนที่ซับซ้อนให้กลายเป็นผลบวกของเศษส่วนที่ง่ายกว่า

เงื่อนไข: ดีกรีของ P(x) ต้องน้อยกว่าดีกรีของ Q(x) (ถ้าไม่เป็นเช่นนั้น ต้องทำการหารยาวก่อน)

รูปแบบต่างๆ ของตัวส่วน:

  • ตัวประกอบเชิงเส้นไม่ซ้ำ: \(\frac{1}{(x-a)(x-b)} = \frac{A}{x-a} + \frac{B}{x-b}\)
  • ตัวประกอบเชิงเส้นซ้ำ: \(\frac{1}{(x-a)^3} = \frac{A}{x-a} + \frac{B}{(x-a)^2} + \frac{C}{(x-a)^3}\)
  • ตัวประกอบกำลังสองที่แยกไม่ได้: \(\frac{1}{(x^2+a^2)(x-b)} = \frac{Ax+B}{x^2+a^2} + \frac{C}{x-b}\)

ส่วนที่ 4: การประยุกต์ใช้การอินทิเกรต

นอกจากการหาพื้นที่ใต้กราฟแล้ว การอินทิเกรตยังสามารถนำไปประยุกต์ใช้ในด้านอื่นๆ ได้อีกมากมาย

1. การหาปริมาตรของรูปทรงตันที่เกิดจากการหมุน (Volume of Revolution)

วิธีแบบจาน (Disk Method)

ใช้เมื่อหมุนพื้นที่รอบแกนที่ "ติดกับ" พื้นที่นั้นๆ แนวคิดคือการรวมปริมาตรของแผ่นจานกลมบางๆ

$$ V = \int_{a}^{b} \pi[f(x)]^2 \,dx \quad \text{(หมุนรอบแกน X)} $$

วิธีแบบเปลือกทรงกระบอก (Cylindrical Shell Method)

ใช้เมื่อหมุนพื้นที่รอบแกนที่ "ไม่ติดกับ" พื้นที่นั้นๆ แนวคิดคือการรวมปริมาตรของเปลือกทรงกระบอกบางๆ

$$ V = \int_{a}^{b} 2\pi x f(x) \,dx \quad \text{(หมุนรอบแกน Y)} $$

ตัวอย่างโจทย์:

จงหาปริมาตรที่เกิดจากการหมุนพื้นที่ใต้กราฟ \(y = \sqrt{x}\) จาก \(x=0\) ถึง \(x=4\) รอบแกน x

วิธีทำ (Disk Method):

  1. รัศมี \(r = f(x) = \sqrt{x}\)
  2. ตั้งอินทิกรัล: \(V = \int_{0}^{4} \pi(\sqrt{x})^2 \,dx = \int_{0}^{4} \pi x \,dx\)
  3. คำนวณ: \(\pi \left[ \frac{x^2}{2} \right]\) ประเมินค่าจาก 0 ถึง 4
  4. \(\pi \left[ \frac{4^2}{2} - \frac{0^2}{2} \right] = \pi \left[ \frac{16}{2} \right] = 8\pi\)

คำตอบ: ปริมาตรคือ \(8\pi\) ลูกบาศก์หน่วย

หัวข้อการประยุกต์ใช้อื่นๆ

  • การหาพื้นที่ระหว่างเส้นโค้ง: \(\int_{a}^{b} [f(x) - g(x)] \,dx\) (ฟังก์ชันบน - ฟังก์ชันล่าง)
  • การหาความยาวส่วนโค้ง (Arc Length)
