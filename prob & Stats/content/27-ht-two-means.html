Two-sample Mean Test

บทที่ 27: การทดสอบสมมติฐานผลต่างระหว่างค่าเฉลี่ย (μ₁ − μ₂)

เมื่อมีสองประชากรอิสระและเราต้องการรู้ว่าค่าเฉลี่ยของพวกเขาต่างกันหรือไม่ เราใช้ผลต่างของค่าเฉลี่ยตัวอย่างเป็นหลักฐานเพื่อทดสอบคำกล่าวอ้าง.

1. ตั้งคดีและสมมติฐาน

สิ่งที่สนใจ

ผลต่าง \(μ_1 - μ_2\) คือหัวใจของคดี ไม่ได้สนใจค่าเฉลี่ยรายตัว

สมมติฐานยอดนิยม
  • \(H_0: μ_1 = μ_2\) หรือ \(μ_1 - μ_2 = 0\) (ไม่มีความต่าง)
  • \(H_1\): >, < หรือ ≠ 0 ตามคำถามวิจัย

บทนี้พิจารณากรณีตัวอย่างอิสระและขนาดใหญ่พอ (\(n_1, n_2 ≥ 30\)) เพื่อใช้ Z-test

2. สูตรสถิติทดสอบ

\[ Z = \frac{(\bar{x}_1 - \bar{x}_2) - (μ_1 - μ_2)_0}{\sqrt{\dfrac{s_1^2}{n_1} + \dfrac{s_2^2}{n_2}}} \]

ตัวประกอบ
  • \(\bar{x}_1 - \bar{x}_2\): ผลต่างที่เห็นในตัวอย่าง
  • \((μ_1 - μ_2)_0\): ผลต่างตาม H₀ (มักเป็น 0)
Standard Error

รวมความผันผวนจากทั้งสองกลุ่มโดย บวก ความแปรปรวนที่ปรับด้วยขนาดตัวอย่าง

3. เข้าใจผลลัพธ์

การตีความค่า Z

ค่า Z ที่สูง (ค่าบวกหรือค่าลบมาก) บอกว่าผลต่างที่สังเกตห่างจาก 0 มากจนไม่น่าจะเกิดจากการสุ่ม

เช็กลิสต์
  • ตัวอย่างจากสองกลุ่มเป็นอิสระ
  • n ใหญ่พอให้ใช้ Z ได้ หรือทราบ σ ของประชากร
  • ระบุทิศทางของ H₁ ให้ชัดเจน (สองหางหรือหางเดียว)

4. เครื่องมือคำนวณผลต่างค่าเฉลี่ย

ปรับค่าพารามิเตอร์ของทั้งสองกลุ่มเพื่อดูสถิติทดสอบ พื้นที่ปฏิเสธ และการตัดสินใจ.

α = 0.01

5. ตัวอย่าง: เปรียบเทียบวิธีการสอน

ข้อมูล: กลุ่มใหม่ \(\bar{x}_1 = 85, s_1 = 8, n_1 = 50\); กลุ่มเก่า \(\bar{x}_2 = 81, s_2 = 10, n_2 = 60\); α = 0.01; \(H_1: μ_1 > μ_2\).

คำนวณสถิติ

\[ Z = \frac{(85 - 81) - 0}{\sqrt{8^2/50 + 10^2/60}} = 2.33 \]

ตัดสินคดี
  • Critical value (หางขวา, α = 0.01): z₀.₀₁ = 2.33 → อยู่ที่เส้นพอดี → ปฏิเสธ H₀
  • p-value ≈ 0.0099 < 0.01 → ปฏิเสธ H₀

สรุป: มีหลักฐานว่าคะแนนเฉลี่ยจากวิธีสอนใหม่สูงกว่าวิธีเดิม
