One-sample Mean Test

บทที่ 25: การทดสอบสมมติฐานค่าเฉลี่ยประชากรเดียว (μ)

เมื่อมีคำกล่าวอ้างเกี่ยวกับค่าเฉลี่ยของประชากรหนึ่งกลุ่ม เราจะใช้ข้อมูลตัวอย่างเพื่อพิสูจน์หรือล้มล้าง โดยเลือกใช้ z-test หรือ t-test ตามความรู้เกี่ยวกับส่วนเบี่ยงเบนมาตรฐาน.

1. โครงเรื่อง 5 ขั้นตอน

  1. ตั้งสมมติฐาน: \(H_0: \mu = \mu_0\); \(H_1\) เป็น >, < หรือ ≠ ตามคดี
  2. กำหนดระดับนัยสำคัญ (α) ก่อนเห็นข้อมูล
  3. เลือกสถิติทดสอบให้เหมาะกับข้อมูล (z-test หรือ t-test)
  4. คำนวณค่าสถิติและตีความด้วย critical value หรือ p-value
  5. สรุปคำตัดสินและตีความในบริบทของปัญหา
เลือกใช้ z-test เมื่อ
  • รู้ค่า σ ของประชากร
  • หรือไม่รู้ σ แต่มี n ใหญ่ (≥ 30) จนใช้ s แทนได้
เลือกใช้ t-test เมื่อ
  • ไม่รู้ค่า σ
  • ขนาดตัวอย่างเล็ก (n < 30)
  • ใช้ df = n − 1 ในการเปิดตาราง t

2. สูตรสถิติทดสอบ

ทั้งสองสูตรมีโครงสร้างเดียวกัน: วัดระยะห่างระหว่างค่าเฉลี่ยตัวอย่างกับค่าที่ H₀ อ้างในหน่วยของความคลาดเคลื่อนมาตรฐาน

z-test

\[ z = \frac{\bar{x} - \mu_0}{\sigma / \sqrt{n}} \]

ใช้การแจกแจงมาตรฐานปกติ

t-test

\[ t = \frac{\bar{x} - \mu_0}{s / \sqrt{n}} \]

ใช้การแจกแจง t ที่ df = n - 1

เมื่อไม่รู้ σ, ส่วนเบี่ยงเบนมาตรฐานของตัวอย่าง (s) ถูกใช้แทน และค่าวิกฤต t จะกว้างกว่าเล็กน้อยเพื่อชดเชยความไม่แน่นอน

3. ทำความเข้าใจการตัดสินใจ

ตรรกะ

ค่าที่คำนวณได้มากหรือน้อยผิดปกติแสดงว่าค่าเฉลี่ยตัวอย่างไกลจาก \(\mu_0\) มากจนไม่น่าใช่เรื่องบังเอิญ → ปฏิเสธ H₀

ข้อควรระวัง
  • ตรวจสอบการสุ่มและความเป็นอิสระของตัวอย่าง
  • สำหรับ t-test ตรวจดูความเป็นปกติ (หรือไม่มี outlier รุนแรง)
  • รายงานผลพร้อมช่วงความเชื่อมั่นหรือขนาดเอฟเฟกต์เสมอ

4. เครื่องคิดเลขทดสอบค่าเฉลี่ย

กรอกข้อมูลตัวอย่าง เลือกประเภทสมมติฐาน ระบบจะคำนวณสถิติทดสอบ p-value และแสดงพื้นที่ปฏิเสธบนกราฟ.

α = 0.05

5. ตัวอย่าง: น้ำหนักขนมปัง

คดี: \(H_0: \mu = 500\) กรัม vs \(H_1: \mu < 500\) กรัม, α = 0.05. สุ่ม 16 ก้อน ได้ \(\bar{x} = 495\), s = 8.

คำนวณสถิติ

t-test (ไม่ทราบ σ, n เล็ก)

\[ t = \frac{495 - 500}{8 / \sqrt{16}} = -2.50 \]

df = 15

เปรียบเทียบ
  • Critical value: t₀.₀₅,15 = -1.753 → -2.50 ต่ำกว่า → ปฏิเสธ H₀
  • p-value ≈ 0.012 < 0.05 → ปฏิเสธ H₀

สรุป: น้ำหนักเฉลี่ยของขนมปังต่ำกว่าที่บริษัทอ้างอย่างมีนัยสำคัญ
