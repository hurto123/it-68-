บทที่ 24: Critical Value vs p-value

สองวิธีในการตัดสินใจจากการทดสอบสมมติฐานที่ให้ผลลัพธ์เหมือนกัน เรียนรู้ข้อดีข้อด้อยและวิธีแปลงไปมาระหว่างกัน.

1. นิยามแบบสั้น

Critical Value Method

กำหนดขอบเขตวิกฤติล่วงหน้า ขึ้นกับ α และรูปแบบ tail จากนั้นเปรียบเทียบสถิติทดสอบกับค่าเกณฑ์

p-value Method

คำนวณความน่าจะเป็นที่สถิติทดสอบจะ “สุดโต่งเท่าหรือมากกว่า” ที่สังเกตได้ แล้วเปรียบเทียบกับ α

ผลลัพธ์ ทั้งสองวิธีให้ข้อสรุปเหมือนกัน หากใช้ α เดียวกันและสถิติทดสอบเดียวกัน

2. ข้อดี-ข้อควรระวัง

Critical Value
  • ง่ายต่อการสื่อสารภาพรวม
  • ใช้ได้ดีสำหรับการคำนวณด้วยมือ/ตาราง
  • ไม่แสดงระดับหลักฐานละเอียดเท่า p-value
p-value
  • ให้ระดับหลักฐานต่อ H₀ โดยตรง
  • เหมาะกับงานวิจัยที่ต้องการรายงานละเอียด
  • ต้องตีความร่วมกับขนาดเอฟเฟกต์

3. เครื่องจำลองความสัมพันธ์

เลือกระดับนัยสำคัญ รูปแบบการทดสอบ และค่าของสถิติทดสอบ (z) เพื่อดูว่าการตัดสินใจแบบ critical value และ p-value ให้ผลตรงกันอย่างไร.

α = 0.05
z = 1.80

4. ตัวอย่างรายงานผล

สถานการณ์ Critical Value p-value สรุป
ทดสอบ \(H_0: \mu = 70\) vs \(H_1: \mu > 70\) z = 2.05 > z₀.₀₅ = 1.645 → ปฏิเสธ H₀ p = 0.020 < 0.05 → ปฏิเสธ H₀ มีหลักฐานว่าค่าเฉลี่ยมากกว่า 70
ทดสอบสองด้าน |t| = 1.90 < t₀.025,28 = 2.048 → ไม่ปฏิเสธ H₀ p = 0.068 > 0.05 → ไม่ปฏิเสธ H₀ ข้อมูลไม่เพียงพอที่จะยืนยันความแตกต่าง

5. Checklist

  • ✅ ระบุ α และรูปแบบ tail ก่อนดูข้อมูล
  • ✅ แสดงทั้งค่า critical และ p-value หากเป็นไปได้
  • ✅ ตีความ p-value ว่าเป็นหลักฐานต่อ H₀ ไม่ใช่ความน่าจะเป็นของสมมติฐาน
  • ✅ ใช้ขนาดเอฟเฟกต์หรือช่วงความเชื่อมั่นประกอบ
