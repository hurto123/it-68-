บทที่ 24: กระบวนการทดสอบสมมติฐาน (Critical Value vs p-value)

เมื่อเราคำนวณค่าสถิติทดสอบเสร็จแล้ว ยังมีอีกสองวิธีในการสรุปคดี: วิธีขอบเขตวิกฤตที่ขีดเส้นตายไว้ล่วงหน้า และวิธี p-value ที่วัดระดับความน่าประหลาดใจของหลักฐาน.

1. จากหลักฐานสู่คำตัดสิน

หัวใจของการทดสอบสมมติฐานคือค่าสถิติทดสอบที่ได้จากสูตรทั่วไป

\[ \text{Test Statistic} = \frac{\text{ค่าที่ได้จากตัวอย่าง} - \text{ค่าที่ H_0 อ้าง}}{\text{ความคลาดเคลื่อนมาตรฐาน}} \]

ค่าสถิตินี้บอกว่าหลักฐานของเราห่างจากสิ่งที่ H₀ เชื่อไว้กี่หน่วยของความไม่แน่นอน ก่อนตัดสินใจ เราต้องเลือกว่าจะตีความค่านี้ด้วยวิธีขอบเขตวิกฤตหรือด้วย p-value.

2. วิธีตัดสินคดี #1: ขอบเขตวิกฤต

ขั้นตอนการทำงาน
  • กำหนดระดับนัยสำคัญ (α) และชนิดของการทดสอบ (หางเดียว/สองหาง)
  • เปิดตารางสถิติ (Z, t, χ², F) เพื่อหา ค่าวิกฤต
  • สร้างพื้นที่ปฏิเสธตามรูปแบบของการทดสอบ
  • คำนวณค่าสถิติทดสอบจากข้อมูลตัวอย่าง
  • หากค่าสถิติตกในพื้นที่ปฏิเสธ → ปฏิเสธ H₀
ข้อสังเกตสำคัญ
  • เห็นภาพได้ง่ายและสะดวกสำหรับการคำนวณด้วยมือ
  • เหมาะกับการสอนและการนำเสนอด้วยกราฟการแจกแจง
  • แต่ไม่บอกระดับความหนักแน่นของหลักฐานแบบละเอียด

3. วิธีตัดสินคดี #2: p-value

p-value คือความน่าจะเป็นที่จะพบค่าสถิติที่สุดโต่งเท่ากับหรือมากกว่าที่เราเจอ โดยอาศัยสมมติว่า H₀ เป็นจริง:

ขั้นตอน
  • กำหนดระดับนัยสำคัญ (α) ล่วงหน้า
  • คำนวณค่าสถิติทดสอบและหา p-value (มักใช้โปรแกรม)
  • เปรียบเทียบ p-value กับ α
  • ถ้า p-value ≤ α → ปฏิเสธ H₀ (หลักฐานสุดโต่งเกินกว่าที่รับได้)
  • ถ้า p-value > α → ยังไม่ปฏิเสธ H₀
ข้อสังเกต
  • เป็นตัวเลขที่รายงานได้สะดวกในงานวิจัย
  • ช่วยเปรียบเทียบความหนักแน่นของหลักฐานระหว่างงานต่างๆ
  • ต้องตีความร่วมกับขนาดเอฟเฟกต์และบริบท

คำจำง่าย: If the p is low, the null must go! ถ้า p-value ต่ำกว่าเกณฑ์ แปลว่าหลักฐานสุดโต่งเกินกว่าจะปล่อย H₀ ไว้ได้

เครื่องมือเปรียบเทียบสองวิธี

เลือกค่า α รูปแบบหาง และค่าสถิติ z เพื่อดูผลสรุปทั้งสองวิธีพร้อมกราฟพื้นที่ปฏิเสธ.

α = 0.05
z = -1.80

4. เปรียบเทียบภาพรวม

หัวข้อ วิธี Critical Value วิธี p-value
สิ่งที่เปรียบเทียบ ค่าสถิติทดสอบกับค่าวิกฤต p-value กับระดับนัยสำคัญ (α)
หน่วย หน่วยเดียวกับสถิติที่ใช้ (เช่น Z, t) ตัวเลขระหว่าง 0 และ 1 (ความน่าจะเป็น)
มุมมอง ตั้งเส้นแบ่งพื้นที่ที่ไม่ยอมรับ H₀ วัดความสุดโต่งของหลักฐานจากมุมมองของ H₀
ผลลัพธ์สุดท้าย ให้คำตัดสินเหมือนกันเสมอ หากใช้ค่าสถิติเดียวกันและ α เดียวกัน

5. ตัวอย่าง: ทดสอบยาแก้ปวด

คดี: บริษัทอ้างว่ายาออกฤทธิ์เฉลี่ย 15 นาที (\(H_0: \mu = 15\)). เราสุ่มผู้ป่วยมา 36 คน พบค่าเฉลี่ย 14.1 นาที ส่วนเบี่ยงเบนมาตรฐาน 3 นาที ต้องการทดสอบแบบสองหางที่ α = 0.05.

สถิติทดสอบ

เพราะทราบ σ (หรือ n ใหญ่): ใช้ Z-test

\[ Z_{\text{test}} = \frac{14.1 - 15}{3 / \sqrt{36}} = -1.80 \]

คำตัดสิน
  • Critical value: Z₀.₀₂₅ = ±1.96 → -1.80 ไม่ข้ามเส้น → ยังไม่ปฏิเสธ H₀
  • p-value: 2 × P(Z < -1.80) = 0.0718 → มากกว่า 0.05 → ยังไม่ปฏิเสธ H₀

สรุป: หลักฐานยังไม่มากพอที่จะบอกว่ายาออกฤทธิ์ต่างจาก 15 นาที
