เฉลย Bootstrap

กลับแบบฝึก
  • ค่าเฉลี่ยตัวอย่าง = 12.5, ค่า median ≈ 12.55
  • Bootstrap mean (B = 2,000) ให้ percentile 95% CI ≈ [12.25, 12.75], basic CI ใกล้เคียงและ bias น้อย
  • SE จาก bootstrap ≈ 0.127, ส่วน CI แบบ t-classic ≈ [12.19, 12.81] กว้างกว่านิดหน่อยเพราะ n เล็ก
  • หากใช้ median สถิติจะมีช่วงกว้างขึ้น ([12.15, 12.80]) สะท้อนความไวต่อการกระจายไม่สมมาตร
