การถดถอยเชิงเส้นอย่างง่าย (Simple Linear Regression)

กลับหน้าต่อยอด

องค์ประกอบหลัก

  • สมการ \hat{y} = \beta_0 + \beta_1 x ใช้คาดการณ์ตัวแปรตอบสนองจากตัวแปรอิสระตัวเดียว
  • ค่า β̂1 = ความชัน, β̂0 = จุดตัดแกน y
  • ใช้วิธี OLS (Ordinary Least Squares) เพื่อลดผลรวมกำลังสองของ residual

การประมาณค่าและค่าความคลาดเคลื่อน

  • \beta_1 = \frac{\sum (x_i-\bar{x})(y_i-\bar{y})}{\sum (x_i-\bar{x})^2}, \beta_0 = \bar{y} - \beta_1 \bar{x}
  • ส่วนเบี่ยงเบนมาตรฐานของสัมประสิทธิ์: SE(\beta_1) = \frac{s}{\sqrt{\sum (x_i-\bar{x})^2}}
  • ทดสอบสมมติฐาน H_0: \beta_1 = 0 ด้วย t-test และสร้างช่วงความเชื่อมั่น

การประเมินแบบจำลอง

  • ค่า R² = สัดส่วนความแปรปรวนของ y ที่อธิบายด้วย x (0–1)
  • ตรวจ residual plot → ควรสุ่มและไม่มีลักษณะโค้งหรือ funnel
  • ค่า RMSE หรือ MAE ใช้ตีความความคลาดเคลื่อนในหน่วยจริง

การวินิจฉัยสมมติฐาน

  • ใช้ QQ-plot หรือ Shapiro–Wilk ตรวจความปกติของ residual เมื่อ n เล็ก
  • ทดสอบ homoscedasticity ด้วย Breusch–Pagan หรือ White test หากพบ funnel
  • Durbin–Watson ใช้สำหรับข้อมูลอนุกรมเวลาเพื่อจับ autocorrelation
  • หากสมมติฐานไม่ผ่าน → พิจารณาแปลงตัวแปร (log/Box-Cox) หรือใช้ robust regression

สมมติฐานสำคัญ

  • ความสัมพันธ์เชิงเส้นระหว่าง x และ y
  • ความเป็นอิสระของ residual (ตรวจด้วย Durbin-Watson สำหรับข้อมูลอนุกรมเวลา)
  • ความแปรปรวนคงที่ (Homoscedasticity) และความปกติของ residual

แนวทางปฏิบัติ

  • แยกชุดข้อมูล train/test หรือใช้ cross-validation เมื่อข้อมูลมากพอ
  • ปรับมาตรการวัด x เมื่อมีสเกลใหญ่เพื่อให้อินเตอร์เซปต์อ่านง่าย
  • นำเข้า Regression Playground เพื่อทดลองจำลอง
  • เมื่อ pattern ไม่เป็นเส้นตรง → พิจารณาแปลง log หรือ polynomial regression
