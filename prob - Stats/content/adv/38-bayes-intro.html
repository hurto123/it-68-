บทนำสู่การวิเคราะห์แบบเบย์ (Bayesian Inference)

กลับหน้าต่อยอด

หลักคิดพื้นฐาน

  • ใช้ Prior (ความเชื่อก่อนเห็นข้อมูล) ผสานกับ Likelihood เพื่อได้ Posterior
  • ปรับปรุงความเชื่อทุกครั้งที่มีข้อมูลใหม่เข้ามา
  • สามารถเปรียบเทียบกับการอนุมานแบบ frequentist เช่น CI vs Credible Interval

Beta-Binomial model

  • ใช้กับข้อมูลความสำเร็จ/ล้มเหลว: Prior ~ Beta(α, β)
  • Posterior = Beta(α + success, β + failure)
  • Credible interval (เช่น 95%) มาจาก quantile ของ Beta posterior

Normal-Normal model

  • สำหรับค่าเฉลี่ยของข้อมูลปกติที่มีความแปรปรวนทราบ: Prior ~ Normal(μ0, σ02)
  • Posterior ~ Normal(μn, σn2) โดย μ_n = \frac{\frac{μ_0}{σ_0^2} + \frac{n\bar{x}}{σ^2}}{\frac{1}{σ_0^2} + \frac{n}{σ^2}}
  • สามารถตีความเป็นการเฉลี่ยแบบถ่วงน้ำหนักระหว่าง prior กับข้อมูล

Credible Interval vs Confidence Interval

  • Credible Interval: โอกาสที่พารามิเตอร์อยู่ในช่วงนี้ (ตาม posterior)
  • Confidence Interval: ขั้นตอนสร้างช่วงที่ครอบพารามิเตอร์ได้ในระยะยาวตาม sampling distribution
  • เมื่อ prior เป็น non-informative และ n ใหญ่ → CI และ CrI มักใกล้เคียงกัน

แนวทางใช้งาน

  • ใช้ Bayes factor หรือ posterior probability ในการตัดสินใจเชิงธุรกิจ
  • ตั้ง prior จากข้อมูลประวัติหรือผู้เชี่ยวชาญ และอัปเดตเมื่อลงทดลองใหม่
  • Bayesian A/B Tool สำหรับจำลอง Beta-Binomial
  • สำหรับโมเดลซับซ้อน → ใช้ MCMC (เช่น Stan, PyMC) และตรวจ convergence ด้วย trace plot
