การถดถอยเชิงเส้นพหุ (Multiple Linear Regression)

กลับหน้าต่อยอด

โครงแบบจำลอง

  • สมการทั่วไป \hat{y} = \beta_0 + \beta_1 x_1 + \beta_2 x_2 + \cdots + \beta_p x_p
  • ใช้เมื่อหลายตัวแปรร่วมกันอธิบายตัวแปรตอบสนอง
  • เพิ่มตัวแปรควบคุมเพื่อป้องกันความสัมพันธ์หลอกจาก confounder

การตีความสัมประสิทธิ์

  • ค่าของ β̂j คือผลกระทบเฉลี่ยของ xj ต่อ y เมื่อคงตัวแปรอื่นไว้คงที่
  • เปลี่ยนหน่วยของ xj จะเปลี่ยนการตีความของสัมประสิทธิ์ทันที
  • ใช้ standardised coefficients (β*) เพื่อเปรียบเทียบอิทธิพลต่างตัวแปร

Dummy variable & interaction

  • ใช้ dummy (0/1) เพื่อแทนหมวดหมู่ → ต้องลบ baseline 1 หมวดเพื่อเลี่ยง dummy trap
  • interaction term เช่น x_1 x_2 อธิบายผลร่วม (เช่น campaign × season)
  • ตรวจดูการเปลี่ยนแปลงของ R² หรือ F-test เมื่อเพิ่ม interaction

Multicollinearity & VIF

  • VIFj = 1/(1−R²j) เมื่อ R²j คือ R² ของการถดถอย xj กับตัวแปรอื่น
  • VIF > 5 (บางอุตสาหกรรมใช้ >10) → สะท้อน multicollinearity สูง ค่าสัมประสิทธิ์ไม่เสถียร
  • แก้ไขด้วยการตัดตัวแปร, รวมตัวแปร (เช่น ดัชนี), หรือใช้ regularization

การตรวจสอบโมเดล

  • Residual plot สำหรับแต่ละ predictor → ตรวจ homoscedasticity และ linearity
  • ตรวจค่าผิดปกติด้วย leverage (hat values) และ Cook’s distance
  • ใช้ adjusted R², AIC, BIC เพื่อเปรียบเทียบแบบจำลองที่มีจำนวนตัวแปรต่างกัน

โคแวเรียนซ์และการตีความร่วม

  • เมทริกซ์โคแวเรียนซ์ของพารามิเตอร์ = σ²(XᵀX)−1 ใช้หาค่า SE และความสัมพันธ์ระหว่าง β
  • ค่าสหสัมพันธ์สูงระหว่างคอลัมน์ของ X → ส่งผลให้ SE ของสัมประสิทธิ์พุ่งสูง
  • พิจารณาใช้ partial regression plot เพื่อสื่อผลของตัวแปรเมื่อควบคุมตัวแปรอื่น
  • การรายงานควรครอบคลุมทั้งค่า β, SE, และ confidence interval เพื่อสะท้อนความไม่แน่นอน
