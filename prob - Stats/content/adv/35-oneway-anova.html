การวิเคราะห์ความแปรปรวนทางเดียว (One-way ANOVA)

กลับหน้าต่อยอด

Concept หลัก

  • ทดสอบว่า ค่าเฉลี่ยของหลายกลุ่ม (k ≥ 3) เท่ากันหรือไม่
  • แยกความแปรปรวนเป็น ระหว่างกลุ่ม (SSB) และ ภายในกลุ่ม (SSW)
  • สมมติฐาน: H0: μ1 = μ2 = … = μk

สูตรและการคำนวณ

  • MSB = SSB/(k-1), MSW = SSW/(N-k)
  • สถิติทดสอบ F = MSB / MSW ~ Fk-1,N-k ภายใต้ H0
  • ค่า p-value เลือกตัดสินใจเทียบกับระดับนัยสำคัญ (เช่น α = 0.05)

สมมติฐานและการตรวจสอบ

  • แต่ละกลุ่มเป็นอิสระและแจกแจงปกติ (หรือกลุ่มใหญ่พอ)
  • ความแปรปรวนใกล้เคียงกัน (homogeneity) → ใช้ Levene หรือ Bartlett test ตรวจสอบ
  • ใช้ residual plot หรือ QQ-plot ตรวจสอบการเบี่ยงเบน

Post-hoc & การสื่อสารผล

  • หาก H0 ถูกปฏิเสธ → ทำ post-hoc เช่น Tukey HSD, Bonferroni เพื่อระบุคู่ที่ต่างกัน
  • รายงานค่าเฉลี่ย, SD, ขนาดตัวอย่างแต่ละกลุ่ม และ effect size (η² หรือ ω²)
  • แสดงผลด้วย boxplot/interval plot เพื่อให้ผู้ฟังเห็นความแตกต่างชัดเจน

ตัวอย่างการใช้งาน

  • ทดสอบสูตรอาหาร 3 แบบเปรียบเทียบผลผลิตเฉลี่ย
  • ประเมินประสิทธิภาพโฆษณา 4 แคมเปญต่อยอดขายรายวัน
  • ใช้ออกแบบการทดลองแบบ balanced เพื่อให้การตีความง่ายขึ้น

ต่อยอด

  • เมื่อละเมิด homogeneity → ใช้ Welch ANOVA หรือ Kruskal-Wallis
  • ขยายเป็น Two-way ANOVA เพื่อดูผลของสองปัจจัยพร้อมกัน
  • ทดลองคำนวณ ANOVA และส่งออกตารางรายงานอัตโนมัติ
