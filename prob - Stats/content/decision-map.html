Decision Map: เลือกวิธีวิเคราะห์

กลับหน้าแรก

1. เป้าหมายการวิเคราะห์

  • อธิบาย/สรุปตัวแปรเดี่ยว → ใช้บท 01–12 + ตาราง
  • ประมาณค่าพารามิเตอร์ → เลือก CI (บท 16–22)
  • ทดสอบสมมติฐาน → บท 23–30 พร้อมเครื่องมือทดสอบ
  • สร้างแบบจำลอง → หัวข้อขั้นสูง 32–39

2. โฟลว์เลือกวิธี

  1. ชนิดตัวแปรตอบ (ต่อเนื่อง, สัดส่วน, การนับ)
  2. จำนวนกลุ่ม (1, 2, มากกว่า 2, จับคู่)
  3. สมมติฐานเกี่ยวกับ σ หรือโครงสร้างความแปรปรวน
  4. ขนาดตัวอย่างและความสมดุล

ใช้ Power & Sample Size เพื่อตรวจความเป็นไปได้

3. ปุ่มลัดเครื่องมือ

  • Z / t / χ² / F Calculators
  • Power Planner
  • Chi-square Tests
  • ANOVA Calculator
  • Correlation Explorer

4. ขยายเส้นทาง

เมื่อผ่านขั้นพื้นฐาน แนะนำให้เรียนรู้ Regression (บท 33–34), ANOVA (บท 35), และ Bootstrap (บท 37) เพื่อพร้อมใช้งานจริง
