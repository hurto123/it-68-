บทที่ 05

ตัวแปรสุ่ม

ภาพรวม

แปลงเหตุการณ์สุ่มให้เป็นค่าตัวเลขเพื่อวิเคราะห์เชิงปริมาณ

  • ความแตกต่างระหว่างตัวแปรสุ่มแบบไม่ต่อเนื่องและต่อเนื่อง
  • นิยามฟังก์ชันมวลความน่าจะเป็น (PMF) และฟังก์ชันความหนาแน่น (PDF)
  • ฟังก์ชันสะสม (CDF) และการหาความน่าจะเป็นจากกราฟ

สูตรและหลักคิดสำคัญ

เครื่องมือพื้นฐานในการหาความน่าจะเป็นของค่าและช่วง

  • คุณสมบัติ PMF: Σ_x p(x) = 1, p(x) ≥ 0
  • คุณสมบัติ PDF: ∫ f(x) dx = 1, f(x) ≥ 0
  • การหาความน่าจะเป็นจาก CDF: P(a ≤ X ≤ b) = F(b) − F(a^- )

ตัวอย่างสถานการณ์

การแจกแจงจำนวนลูกค้าที่มาถึงร้าน (ไม่ต่อเนื่อง) เทียบกับเวลารอคอยลูกค้าถัดไป (ต่อเนื่อง)

เชื่อมต่อการเรียน

  • เงื่อนไขและกฎบวก/คูณ
  • ค่าคาดหมายและความแปรปรวน
  • สัญลักษณ์ที่ใช้บ่อย
