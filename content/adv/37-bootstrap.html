Bootstrap Confidence Interval

กลับหน้าต่อยอด

แนวคิดพื้นฐาน

  • สุ่มตัวอย่างใหม่พร้อมการคืนตัวอย่าง (resampling with replacement) จากข้อมูลที่มี
  • คำนวณสถิติสนใจ (เช่น mean, median, regression coefficient) ทุกครั้ง → ได้ bootstrap distribution
  • ใช้เมื่อตัวอย่างไม่ใหญ่มากหรือสูตร analytic ของ SE/CI ซับซ้อน

ประเภทของ Bootstrap CI

  • Percentile: ใช้ quantile 2.5% และ 97.5% ของ bootstrap distribution
  • Basic: พลิกค่ารอบสถิติจริง (2\hat{\theta} - \theta^*_{upper}, 2\hat{\theta} - \theta^*_{lower})
  • BCa: ปรับ bias และ skewness ด้วยค่า acceleration (ต้องคำนวณ jackknife)

ขั้นตอนปฏิบัติ

  1. เตรียมข้อมูลตัวอย่าง n ตัว และเลือกสถิติที่ต้องการประเมิน
  2. ทำซ้ำ B ครั้ง (เช่น 1,000 หรือ 10,000): สุ่มตัวอย่าง n ตัวพร้อมคืน → คำนวณสถิติ
  3. ใช้ distribution ที่ได้สรุปค่าเฉลี่ย, SE, หรือ percentile สำหรับสร้าง CI

ข้อควรระวัง

  • ข้อมูลควรเป็นตัวแทนของประชากร หากมี bias จะถูกขยายซ้ำใน bootstrap
  • ข้อมูลที่มีโครงสร้างพิเศษ (อนุกรมเวลา, คลัสเตอร์) → ใช้ block bootstrap หรือ cluster bootstrap
  • จำนวนรอบ B สูงขึ้นช่วยให้ CI เสถียร แต่ต้องคำนึงถึงเวลา/ทรัพยากรคอมพิวเตอร์

ใช้งานร่วมกับเครื่องมือ

  • ใช้ web worker ในหน้า Bootstrap Lab เพื่อรันการคำนวณจำนวนมากแบบขนาน
  • สื่อสารผลด้วย histogram ของ bootstrap statistic และสรุปค่า CI อย่างชัดเจน
