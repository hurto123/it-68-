CI ของค่าเฉลี่ยเมื่อรู้ \(\sigma\)

ใช้ Z-interval เมื่อเราทราบส่วนเบี่ยงเบนมาตรฐานของประชากรหรือมีข้อมูลเชื่อถือได้จากอดีต

\[ CI_{\mu} = \bar X \pm Z_{\alpha/2}\,\frac{\sigma}{\sqrt{n}} \]

เป้าหมาย

คำนวณช่วงความเชื่อมั่นของค่าเฉลี่ยประชากรเมื่อ \(\sigma\) เป็นที่ทราบ (เช่น มาตรฐานโรงงาน)

สัญลักษณ์

\(\bar X\)
ค่าเฉลี่ยตัวอย่าง
\(\sigma\)
ส่วนเบี่ยงเบนมาตรฐานของประชากร (ทราบค่า)
\(n\)
ขนาดตัวอย่าง
\(Z_{\alpha/2}\)
ค่าวิกฤตจากการแจกแจงปกติมาตรฐาน
\(\alpha\)
นัยสำคัญ (1 − ระดับความเชื่อมั่น)

วิธีทำ

  1. ตรวจสอบว่าการสุ่มเป็นอิสระ และการแจกแจงของตัวอย่างใกล้ปกติหรือมี \(n \ge 30\)
  2. กำหนดระดับความเชื่อมั่น → หา \(Z_{\alpha/2}\)
  3. คำนวณ \(SE = \sigma/\sqrt{n}\)
  4. แทนค่าสูตร \(CI = \bar X \pm Z_{\alpha/2} SE\) และสรุปผล

ตัวอย่าง

โรงงานบรรจุกาแฟเคยวัดได้ \(\sigma = 0.4\) กรัม สุ่มซองใหม่ 45 ชิ้นได้ \(\bar X = 12.1\) กรัม ต้องการ CI 95%

  1. \(SE = 0.4/\sqrt{45} = 0.0596\)
  2. 95% ⇒ \(Z_{0.025} = 1.96\)
  3. \(ME = 1.96 \times 0.0596 = 0.117\)
  4. ช่วง = 12.1 ± 0.117 ⇒ (11.983, 12.217)

ข้อควรระวัง

  • ถ้าค่า \(\sigma\) ที่นำมาใช้ล้าสมัยหรือไม่ตรงสถานการณ์ ควรใช้ t-interval แทน
  • เมื่อ \(n\) เล็ก (< 30) ต้องมั่นใจว่าข้อมูลใกล้ปกติจึงจะใช้สูตรนี้ได้

CI ของค่าเฉลี่ยเมื่อไม่รู้ \(\sigma\)

ใช้ t-interval พร้อมองศาอิสระ \(df = n-1\) เมื่อส่วนเบี่ยงเบนมาตรฐานของประชากรไม่ทราบ

\[ CI_{\mu} = \bar X \pm t_{\alpha/2,\,n-1}\,\frac{s}{\sqrt{n}} \]

เป้าหมาย

ประเมินค่าเฉลี่ยประชากรโดยใช้ \(s\) เป็นตัวแทน \(\sigma\) และคำนึงถึงความไม่แน่นอนที่เพิ่มขึ้น

สัญลักษณ์

\(s\)
ส่วนเบี่ยงเบนมาตรฐานของตัวอย่าง
\(t_{\alpha/2, df}\)
ค่าวิกฤตจาก t-distribution, \(df = n-1\)
อื่น ๆ
เหมือนกรณีรู้ \(\sigma\): \(\bar X, n, \alpha\)

วิธีทำ

  1. ยืนยันการสุ่มอิสระ และตรวจสอบความเป็นปกติ (กรณี \(n < 30\))
  2. เลือกระดับความเชื่อมั่น → หา \(t_{\alpha/2, n-1}\)
  3. คำนวณ \(SE = s/\sqrt{n}\)
  4. สร้างช่วงด้วยสูตร t-interval และสื่อสารผล

ตัวอย่าง

สุ่มนักศึกษาจำนวน 16 คนพบเวลาอ่านหนังสือเฉลี่ย \(\bar X = 45\) นาที, \(s = 12\) นาที ต้องการ CI 95%

  1. \(SE = 12/\sqrt{16} = 3\)
  2. \(df = 15\) ⇒ \(t_{0.025,15} ≈ 2.131\)
  3. \(ME = 2.131 \times 3 = 6.393\)
  4. CI = 45 ± 6.393 ⇒ (38.607, 51.393)

ข้อควรระวัง

  • เมื่อ \(n\) ใหญ่ t-distribution จะเข้าใกล้ Z ใช้สูตรไหนผลแทบเท่ากัน
  • ข้อมูลที่เบ้รุนแรงหรือมี outlier มาก ควรใช้เทคนิคปรับก่อนสร้าง CI
