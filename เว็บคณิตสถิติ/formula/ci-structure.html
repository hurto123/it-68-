โครงสร้าง Confidence Interval

Point Estimate ± Critical Value × Standard Error — หัวใจของบทที่ 12–16

อัปเดต 7 ต.ค. 2025 บทที่เกี่ยวข้อง: 11–16
เปิดบทเรียน ดูสรุป

สารบัญ

  1. โครงสร้างช่วงความเชื่อมั่น (CI Framework)

โครงสร้างช่วงความเชื่อมั่น

ใช้กับพารามิเตอร์ใดก็ได้เมื่อเรามี Estimator, ค่าช่องผิดพลาดมาตรฐาน และการแจกแจงอ้างอิง

\[ CI = \text{Point Estimate} \pm (\text{Critical Value}) \times (\text{Standard Error}) \]

เป้าหมาย

สร้างช่วงค่าที่เชื่อถือได้สำหรับพารามิเตอร์ของประชากรจากข้อมูลตัวอย่าง โดยสื่อสารความไม่แน่นอนอย่างโปร่งใส

สัญลักษณ์

\(\hat\theta\)
Point estimate ที่ได้จากตัวอย่าง เช่น \(\bar X\), \(\hat p\)
CV
Critical Value จาก Z หรือ t ตามระดับความเชื่อมั่น
SE
Standard Error ของ Estimator ขึ้นกับรูปแบบข้อมูลและขนาดตัวอย่าง
CL
Confidence Level (เช่น 90%, 95%, 99%)

วิธีทำ

  1. ระบุพารามิเตอร์และ Estimator ที่จะใช้ พร้อมตรวจสอบว่าข้อสมมติพื้นฐานเป็นจริง
  2. เลือกการแจกแจงอ้างอิง (Z, t, หรือการแจกแจงเฉพาะอื่น) แล้วดึงค่า CV ที่ตรงกับระดับความเชื่อมั่น
  3. คำนวณ SE ด้วยสูตรของ Estimator นั้น ๆ
  4. แทนค่าในสมการ \(CI = \hat\theta \pm CV \times SE\) และสรุปผลด้วยภาษาคน

ตัวอย่าง

สุ่มกาแฟ 45 ซองได้ \(\bar X = 12.1\) กรัม, \(\sigma = 0.4\) กรัม ต้องการ CI 95%

  1. \(SE = 0.4/\sqrt{45} = 0.0596\)
  2. ระดับ 95% → \(CV = Z_{0.025} = 1.96\)
  3. \(ME = 1.96 \times 0.0596 = 0.117\)
  4. CI = \(12.1 \pm 0.117\) ⇒ (11.983, 12.217)

สรุป: เรามั่นใจ 95% ว่าค่าเฉลี่ยปริมาณกาแฟต่อซองอยู่ในช่วง 11.98–12.22 กรัม

ข้อควรระวัง

  • ความเชื่อมั่นสูงขึ้น → CV สูงขึ้น → ช่วงกว้างขึ้น ต้องเพิ่ม \(n\) หากอยากได้ช่วงแคบ
  • อย่าตีความว่า “ความน่าจะเป็น 95% ที่ค่าจริงอยู่ในช่วง” เพราะค่าจริงคงที่ ช่วงต่างหากที่เปลี่ยนได้
  • ตรวจสอบข้อสมมติเกี่ยวกับการสุ่มอิสระและความเป็นปกติ (หรือใช้กฎ \(n \ge 30\)) ก่อนใช้สูตรอนุพันธ์จาก CLT
