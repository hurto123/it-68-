Confidence Intervals

เริ่มจากโครงสร้าง CI แล้วเลือกสูตรเฉพาะสถานการณ์เพื่อคุมระดับความเชื่อมั่นและความแม่นยำ

หัวข้อ สูตร ใช้เมื่อ อ้างอิง
โครงสร้าง CI \(\text{Point Estimate} \pm (\text{Critical Value})(\text{SE})\) ทุก CI ใช้โครงสร้างนี้ • คุมความเชื่อมั่นผ่านค่าวิกฤต • ลดความกว้างด้วยการเพิ่ม \(n\) CI Anatomy
ค่าเฉลี่ย (รู้ \(\sigma\)) \(\bar X \pm Z_{\alpha/2}\dfrac{\sigma}{\sqrt{n}}\) ทราบ \(\sigma\) จากมาตรฐานหรือข้อมูลอดีต • ตัวอย่าง \(n \ge 30\) หรือใกล้ปกติ CI Mean (Z)
ค่าเฉลี่ย (ไม่รู้ \(\sigma\)) \(\bar X \pm t_{\alpha/2,df}\dfrac{s}{\sqrt{n}}\) ใช้ \(s\) แทน \(\sigma\) • ต้องอาศัย t-distribution ด้วย \(df = n-1\) CI Mean (t)
สัดส่วน \(\hat p \pm Z_{\alpha/2}\sqrt{\dfrac{\hat p(1-\hat p)}{n}}\) เงื่อนไขเพียงพอ: \(n\hat p, n(1-\hat p) \ge 10\) • ใช้ p* = 0.5 เมื่อออกแบบล่วงหน้า CI Proportion
ผลต่างค่าเฉลี่ย \((\bar X_1-\bar X_2) \pm t_{\alpha/2,df} S_p\sqrt{\tfrac{1}{n_1}+\tfrac{1}{n_2}}\) สองกลุ่มอิสระ • สมมติ \(\sigma_1^2 = \sigma_2^2\) • ใช้ pooled variance CI Difference (Mean)
Pooled variance \(S_p^2 = \dfrac{(n_1-1)s_1^2 + (n_2-1)s_2^2}{n_1+n_2-2}\) ใช้คำนวณ SE ใน CI/ทดสอบค่าเฉลี่ยสองกลุ่มเมื่อสมมติความแปรปรวนเท่ากัน Pooled Var
ผลต่างสัดส่วน \((\hat p_1-\hat p_2) \pm Z_{\alpha/2}\sqrt{\dfrac{\hat p_1(1-\hat p_1)}{n_1}+\dfrac{\hat p_2(1-\hat p_2)}{n_2}}\) เปรียบเทียบ CTR/อัตราความสำเร็จสองแคมเปญ • ตรวจดูว่า 0 อยู่ในช่วงหรือไม่ CI Difference (Proportion)
Tip: หาก CI กว้างเกินไปแต่ต้องการคงระดับความเชื่อมั่น ให้เพิ่มขนาดตัวอย่าง (ดูส่วน Sample Size ด้านล่าง)

Sample Size Planner

กำหนดจำนวนตัวอย่างที่ต้องเก็บเพื่อให้ช่วงความเชื่อมั่นแคบตามงบและเวลาที่มี

เป้าหมาย สูตร หมายเหตุ อ้างอิง
ค่าเฉลี่ย \(n = \left(\dfrac{Z_{\alpha/2}\,\sigma}{E}\right)^2\) ต้องประมาณ \(\sigma\) จากข้อมูลเดิม, pilot หรือกฎ range/4 Sample Size — Mean
สัดส่วน \(n = \dfrac{Z_{\alpha/2}^2 p(1-p)}{E^2}\) ไม่รู้ \(p\) ให้ใช้ 0.5 (อนุรักษ์นิยม) • ปรับ FPC หากประชากรเล็ก Sample Size — Proportion
Checklist: ระบุระดับความเชื่อมั่น, กำหนด \(E\) (ความกว้างครึ่งหนึ่งของ CI) และตรวจสอบงบ/เวลาให้สอดคล้องกับ \(n\) ที่คำนวณได้

One-sample Tests

สูตรทดสอบสมมติฐานสำหรับค่าเฉลี่ยและสัดส่วนเมื่อมีข้อมูลเพียงหนึ่งประชากร

สถิติทดสอบ สูตร ใช้เมื่อ อ้างอิง
Z-test ค่าเฉลี่ย \(Z = \dfrac{\bar X - \mu_0}{\sigma/\sqrt{n}}\) รู้ \(\sigma\) หรือ \(n \ge 30\) พร้อมแจกแจงใกล้ปกติ Mean Tests
t-test ค่าเฉลี่ย \(t = \dfrac{\bar X - \mu_0}{s/\sqrt{n}}\) ไม่รู้ \(\sigma\) • ใช้ \(df = n-1\) • ตรวจสอบความเป็นปกติ/ไม่มี outlier หนัก Mean Tests
Z-test สัดส่วน \(Z = \dfrac{\hat p - p_0}{\sqrt{\dfrac{p_0(1-p_0)}{n}}}\) เงื่อนไขเพียงพอ: \(np_0, n(1-p_0) \ge 10\) • ใช้ในการทดสอบ Claim ของอัตรา Proportion Test
จำไว้ว่า: ตัดสินใจโดยเทียบสถิติกับค่าวิกฤตหรือหาค่า p-value และระบุนัยสำคัญทางปฏิบัติร่วมด้วย ไม่ใช่แค่ผลทางสถิติ

Two-sample Comparisons

เลือกสูตรตามประเภทตัวแปรและสมมติฐานความแปรปรวน

สถานการณ์ สูตร หมายเหตุ อ้างอิง
t-test ค่าเฉลี่ย (pooled) \(t = \dfrac{(\bar X_1-\bar X_2) - (\mu_1-\mu_2)_0}{S_p \sqrt{\tfrac{1}{n_1}+\tfrac{1}{n_2}}}\) ต้องใช้ \(S_p\) จากสูตร pooled variance • \(df = n_1 + n_2 - 2\) Two-sample Tests
Pooled variance \(S_p^2 = \dfrac{(n_1-1)s_1^2 + (n_2-1)s_2^2}{n_1+n_2-2}\) ใช้กับทั้ง CI และการทดสอบเมื่อสมมติความแปรปรวนเท่ากัน Two-sample Tests
Z-test ผลต่างสัดส่วน \(Z = \dfrac{(\hat p_1-\hat p_2) - (p_1-p_2)_0}{\sqrt{\dfrac{\hat p_1(1-\hat p_1)}{n_1}+\dfrac{\hat p_2(1-\hat p_2)}{n_2}}}\) นิยมตั้ง \((p_1-p_2)_0 = 0\) เพื่อตรวจว่าต่างกันหรือไม่ • ใช้เงื่อนไขเพียงพอแบบเดียวกับ CI สัดส่วน Two-sample Tests
Note: หากหลักฐานชี้ว่าความแปรปรวนไม่เท่ากันหรือขนาดตัวอย่างต่างกันมาก ให้ใช้ Welch t-test (ดูรายละเอียดในสรุปบทที่ 21)

Variance & Consistency Tests

ใช้การแจกแจง \(\chi^2\) และ F เพื่อประเมินเสถียรภาพของกระบวนการ

ทดสอบ สูตร ใช้เมื่อ อ้างอิง
Chi-square test \(\chi^2 = \dfrac{(n-1)s^2}{\sigma_0^2}\) ตรวจสอบว่าความแปรปรวนของกระบวนการเปลี่ยนไปจากมาตรฐานหรือไม่ • \(df = n-1\) Variance Tests
F-test \(F = \dfrac{s_1^2}{s_2^2}\) เปรียบเทียบสองความแปรปรวน • จัดให้ \(s_1^2\) เป็นค่าที่มากกว่าเสมอเพื่อให้ \(F \ge 1\) Variance Tests
ข้อควรระวัง: ทดสอบเหล่านี้ไวต่อการเบ้ของข้อมูล • หากแจกแจงไม่ใกล้ปกติควรใช้วิธี nonparametric หรือแปลงข้อมูลก่อน

Quick Workflow & Tips

ก่อนเริ่ม

  1. ระบุชนิดข้อมูล (เชิงปริมาณ/สัดส่วน) และจำนวนกลุ่ม
  2. ตรวจสมมติฐาน (การสุ่ม, ความเป็นอิสระ, รูปร่างแจกแจง)
  3. ตั้งเป้าหมาย: สร้าง CI หรือทดสอบสมมติฐาน

ระหว่างคำนวณ

  1. เลือกสูตรให้ถูกกับสถานการณ์ (ตารางด้านบน)
  2. แทนค่าอย่างมีขั้นตอน จด ME/SE แยกต่างหากเพื่อ debug ง่าย
  3. ใช้เครื่องคิดเลข/สเปรดชีตช่วยเพื่อลดข้อผิดพลาดจากการป้อนค่า

หลังได้ผล

  1. แปลผลเป็นภาษาที่ Stakeholder เข้าใจ (ช่วง, ความต่าง, นัยสำคัญ)
  2. แนบข้อจำกัด (สมมติฐาน, ขนาดตัวอย่าง, คุณภาพข้อมูล)
  3. แนะนำการเก็บข้อมูลเพิ่มหรือการวิเคราะห์ต่อยอดหากจำเป็น
