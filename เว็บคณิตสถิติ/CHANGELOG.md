# Changelog — เว็บคณิตสถิติ

# 2025-09-24
- เปิดตัวหน้าแบบฝึกหัดและ Quiz รายบท (บทที่ 1–9) พร้อมข้อสอบรวมทุกบท มีโหมดฝึก/จับเวลาและสรุปผลหลังทำ
